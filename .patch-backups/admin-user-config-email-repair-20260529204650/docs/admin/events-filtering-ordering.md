# Events filtering and ordering

The tenant admin Events page supports search, type filtering, status filtering, sort selection, and ascending/descending order.

Admins can set a display/order number on each event. The default list order is display/order number ascending, then newest record ID descending.

Use `/admin/events` to filter events and `/admin/events/edit` to add or edit the order number, status, date, type, location, work name, and notes.
