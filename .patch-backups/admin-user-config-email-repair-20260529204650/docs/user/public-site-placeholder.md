# Public Site Placeholder Behavior

## Current behavior

Tenant public sites currently render placeholder pages while the platform HTTP routing layer is being established.

Current tenant pages:

```text
Home
Portfolio
About
Contact
```

## Expected future behavior

Tenant pages will use tenant settings, artwork records, portfolio sections, media assets, and theme settings.

## Current limitation

The existing bxiie.com visual design has not yet been wired into the new HTTP entrypoint.
