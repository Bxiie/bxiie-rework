# Contact Messages and Email Signups Administration

## Current records

```text
contact_messages
email_signups
```

## Current notification behavior

Contact messages and signups queue tenant admin notification emails when:

```text
tenant_settings.site_admin_email
```

is set.

## Production requirements

```text
admin UI for contact messages
admin UI for email signups
CSV export
spam filtering
reCAPTCHA or equivalent
consent tracking
unsubscribe flow
privacy/delete workflow
```
