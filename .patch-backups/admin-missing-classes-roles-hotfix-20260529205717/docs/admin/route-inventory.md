# Route Inventory for Administrators

## Tenant domains

Tenant admins sign in at:

```text
https://tenant-domain/login
```

The root URL is public site content:

```text
https://tenant-domain/
```

## Platform domain

Platform operators use the platform domain for platform-wide admin tools:

```text
https://artsfol.io/admin
https://artsfol.io/admin/jobs
https://artsfol.io/admin/workers
```

Tenant domains must not expose platform job or worker screens.
