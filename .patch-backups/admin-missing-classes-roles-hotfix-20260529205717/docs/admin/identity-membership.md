# Identity and Membership Administration

ArtsFolio supports:

```text
OAuth/OIDC
Local email/password
```

Platform roles:

```text
owner
admin
support
readonly
```

Tenant roles:

```text
owner
admin
editor
viewer
```

Tenant admins manage tenant invitations, tenant user roles, tenant access review, and tenant account/billing details.
