# ArtsFolio plans

ArtsFolio plans define the number of artwork records, email-list addresses, domain options, and support level available to an artist site.

The public pricing page shows:

- monthly plan amount;
- included artwork and email-list limits;
- whether custom domains are included;
- whether an ArtsFolio notification/link appears on free tenant pages;
- ArtsFolio commission on platform-processed sales.

Tenant admins can review their current plan from **Tenant Admin → Billing**.
