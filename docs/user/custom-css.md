# Custom CSS

Click **Settings** in your admin sidebar to edit Custom CSS.

A new site begins with a commented copy of the public stylesheet, including a
selector guide and a section for your own additions. Put new overrides near the
bottom, save one change at a time, and review desktop and phone layouts.

ArtsFolio does not automatically replace your Custom CSS when shared styles
change. Your edits remain yours.
