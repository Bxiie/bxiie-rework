# Signup code visibility

Platform admins can choose whether the signup-code page shows old codes.

By default, the list shows active codes and hides:

- used codes
- revoked codes

Use **Signup code list options** on the signup-code page to show or hide either category. ArtsFolio remembers the choice in that browser.

A used code has already reached its redemption limit. A revoked code was manually disabled by a platform admin. Neither can be used to create a site or apply free plan access.
