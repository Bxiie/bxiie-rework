# Color palettes

Use **Tenant Admin → Settings → Colors and background** to change your site's colors.

The palette buttons at the top of the section provide quick starting points. Choose a palette first, then adjust any individual color picker, opacity field, or background setting before saving.

The first palette, **Default**, restores the standard ArtsFolio new-site color direction.

Palette buttons do not replace your selected background images. Use the existing image pickers in the same section to choose page, menu, top bar, or artwork-card background images.
