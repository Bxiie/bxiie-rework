# Upload Artwork

Tenant admins can upload first artwork images at:

```text
/admin/artwork/upload
```

This first version stages uploads for later publishing.

<!-- End of file. -->

## Artwork fields

The upload form collects:

```text
title
date/year
medium
notes
sale status
price
image
```

New artwork starts as a draft so it can be reviewed before publishing.
