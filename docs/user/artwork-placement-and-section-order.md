# Placing artwork on the home page and in sections

Open **Admin → Artworks → Artwork Placement Matrix** to quickly assign artwork to the home page and portfolio sections. Each row shows the artwork thumbnail and checkboxes for each destination.

Open **Admin → Portfolio Sections → Order artwork in sections and home page** to change the display order. Drag rows into the desired order, then save.

Password reset links are sent only to email addresses that are attached to users for the current artist site.
