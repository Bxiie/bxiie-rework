# Contact forms and visual settings

The public Contact page uses the same visual settings as the rest of the tenant site. Form panels, labels, inputs, CAPTCHA, and buttons inherit the tenant text color, content background, and text spread settings.

The mailing-list signup remains available from the tenant footer on public pages. The older delayed signup popup is hidden so visitors see one clear signup path instead of duplicate prompts.

In Tenant Admin > Settings, the page preview uses the tenant top bar, menu, heading, content, text, and artwork-card settings. Adjust colors, opacity, background images, and drop shadows there to tune both the public pages and the admin preview.
