# Site branding images

Tenant Settings can use published Site Images for public page backgrounds and the top bar background.

The top bar supports a background color, a Site Image background, and opacity. The public site header and tenant admin header both use the same configured top bar color/image values.

Only published artwork marked as Site Images appears in site-image pickers. Images can also be marked Portfolio Images when they should appear in public portfolio pages.
