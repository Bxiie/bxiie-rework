# Using Site Images

Some images are part of your portfolio. Other images are just useful for page backgrounds, About page decoration, or Contact page decoration.

When editing an artwork, choose:

- `Portfolio Images` when the image should appear in your public artwork pages.
- `Site Images` when the image should be available for About, Contact, and Background choices.
- Both when the image should do both jobs.

About, Contact, and Background image controls only show published Site Images.
