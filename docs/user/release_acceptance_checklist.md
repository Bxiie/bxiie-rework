# ArtsFolio release acceptance checklist

Use this shorter checklist when accepting a release from the artist or operator perspective.

| Check | Pass | Fail | N/A | Notes |
| --- | --- | --- | --- | --- |
| I can sign in and sign out. | ☐ | ☐ | ☐ | |
| A bad password shows a branded retry page. | ☐ | ☐ | ☐ | |
| My public site looks correct on desktop. | ☐ | ☐ | ☐ | |
| My public site looks correct on phone. | ☐ | ☐ | ☐ | |
| Anonymous visitors do not see an Admin tab. | ☐ | ☐ | ☐ | |
| Logged-in admins see the Admin tab. | ☐ | ☐ | ☐ | |
| Dashboard numbers match my artwork, sales, subscribers, messages, and traffic. | ☐ | ☐ | ☐ | |
| I can upload and edit artwork. | ☐ | ☐ | ☐ | |
| I can change settings and see them on the site. | ☐ | ☐ | ☐ | |
| Contact messages and email signups are recorded. | ☐ | ☐ | ☐ | |
| Sales and sales analytics pages load. | ☐ | ☐ | ☐ | |
