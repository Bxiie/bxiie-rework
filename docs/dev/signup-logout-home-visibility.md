# Signup, logout, and home artwork visibility

This update keeps tenant signup, logout, and artwork placement behavior explicit.

- Signup passcodes are validated before the site details form is shown.
- Tenant logout revokes the server-side session and expires browser cookies.
- The artwork list links directly to the placement matrix and section order page.
- The public home page shows only artwork explicitly assigned to the home page when the home assignment table exists.
