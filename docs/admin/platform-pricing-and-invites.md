# Platform pricing and invite administration

Use **Platform Admin → Pricing** to set plan names, prices, allowed artworks, allowed email addresses, custom domain availability, active state, display order, descriptions, and the platform sales commission.

The seeded strawman plans are Free, Studio, Professional, and Collective. The Free plan includes an ArtsFolio notification/link on free tenant pages.

Use **Platform Admin → Users** to send a new platform admin invite or resend an invite to an existing platform admin. Use **Tenant Admin → Users** to send or resend tenant admin invites.

Deleting a tenant requires typing `delete`. The tenant is soft-deleted and removed from the normal tenant list; data is not physically destroyed by this action.
