# Tenant visual surfaces

Tenant owners and tenant admins manage public and admin visual-surface settings from Tenant Admin → Settings.

The shared controls include:

- Default text color.
- Header drop shadow on/off and shadow CSS.
- Top bar background color, image, and opacity.
- Menu background color, image, and opacity.
- Heading spread color and opacity.
- Content background color and opacity.
- Text spread color and opacity.
- Portfolio artwork card background color, Site Image, opacity, and background-size.

Site Image pickers only show published images marked as `Site Images`. Portfolio card backgrounds do not make an image public by themselves; public portfolio visibility still requires the artwork to be marked as `Portfolio Images` and published.

Use high-contrast text colors when setting image-heavy backgrounds. The default text color is `#1f1a14`, a warm near-black intended to read cleanly against the default cream overlays.
