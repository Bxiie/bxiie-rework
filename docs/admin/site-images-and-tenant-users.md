# Site Images and Tenant User Controls

Tenant admins can classify uploaded images as `Portfolio Images`, `Site Images`, or both.

- Use `Portfolio Images` for published work that should appear on the public home, portfolio, portfolio section, and artwork detail pages.
- Use `Site Images` for published images that should appear in About, Contact, and Background image pickers.
- Mark an image as both when it is both public portfolio work and useful as site decoration.

Only published `Site Images` appear in About, Contact, and Background pickers. Pickers show thumbnails so admins can choose visually.

Tenant owners can delete tenant users. The delete control requires typing `delete` in the confirmation prompt. User deletion revokes tenant access and writes an audit log event.
