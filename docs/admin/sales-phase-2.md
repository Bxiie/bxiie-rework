# Sales phase 2 admin guide

Platform admins configure Stripe keys and sales commission in Platform Admin settings/pricing. Tenant admins add their Stripe connected account ID in Tenant Admin Settings.

Tenant sales workflow is available at Tenant Admin → Sales. Orders move through:
1. Ordered
2. Acknowledged
3. Packed
4. Shipped

Shipping carrier, tracking number, and tracking URL can be saved with the order.

Platform Admin → Sales shows tenant sales, workflow status, gross sale totals, and calculated platform commission.
