# Admin notifications and image watermarks

Platform and tenant admin navigation shows count badges for new messages and email signups. Opening the email-signup list records the current view time; message badges track records whose status is new. Tenant Settings > Watermark lets an administrator opt in and select watermark text format, custom text, position, light/dark appearance, opacity, and size. Watermarks are rendered for public medium, large, and original responses without modifying stored originals or admin thumbnails.
