# Public page visibility and mailing-list invitation

Tenant Admin → Settings → Identity includes switches to suppress About, suppress
Contact, remove the Portfolio `All` button, and suppress both the footer signup
form and delayed mailing-list dialog.

The invitation appears after 60 seconds for an unknown signed-out visitor.
Dismissal suppresses it for 30 days; successful signup marks the browser known
for one year.
