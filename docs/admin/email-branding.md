# Email branding

Outbound platform email uses shared ArtsFolio branding automatically. Platform admins do not need to edit each individual email template to add the brand header or footer.

The shared shell appears on password reset, email verification, tenant contact notifications, mailing-list notifications, lifecycle messages, signup-code invitations, and tenant-admin invitations.

If an email appears unbranded, check the email outbox row first. New rows should contain the ArtsFolio header and footer in `body_text`, and a branded HTML companion in `body_html`.
