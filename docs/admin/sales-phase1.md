# Sales readiness controls

Tenant admins can prepare artwork for sales before checkout is enabled.

Use Artworks to:
- mark an artwork as `For sale`, `Sold`, or `NFS`;
- enter a visible price;
- choose whether the item is a one-off artwork or an inventory-backed multiple;
- enter inventory quantity for multiple items.

Use Site settings → Sales notes to explain how sales work for the tenant. These notes appear on artwork detail pages.

Payment note: checkout will be handled through Stripe in the sales order phase. Free plans do not have online checkout.
