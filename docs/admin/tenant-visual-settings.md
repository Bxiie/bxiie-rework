# Tenant visual settings

Tenant visual settings control both public tenant pages and tenant admin pages so administrators preview the same visual system their visitors see.

## Header and navigation

The top bar supports a background color, opacity, image, and optional drop shadow. The menu/nav panel supports its own color, opacity, image, and an on/off setting. Use **Suppress panel** to remove the visible wash around the top navigation while keeping the menu links in place.

## Content and text readability

Heading spread, content area background, and text spread controls are shared by public and admin pages. Use opacity values from `0` to `1`; `0` disables the wash and `1` makes it fully opaque.

## Contact forms

Tenant contact and mailing-list forms use the same surface variables as other tenant content. Successful asynchronous submissions show an inline success message and clear the form.
