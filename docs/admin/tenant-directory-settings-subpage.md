# Tenant directory settings subpage

Tenant directory controls live under Tenant Admin -> Settings -> Directory.

The Directory settings subpage lets tenant admins:

- opt the tenant into or out of the public ArtsFolio directory,
- select a published artwork thumbnail for the directory card, and
- write the short public directory summary.

The old `/admin/directory` URL redirects to `/admin/settings?section=directory` so old bookmarks do not break.

Saving the Directory subpage writes only directory-owned settings and does not touch Identity, Typography, Colors & Backgrounds, Miscellaneous, or Custom CSS settings.


Directory thumbnail selection moved to Tenant Admin -> Artworks. Use the Directory thumbnail checkbox beside an eligible published artwork; the Directory settings subpage only displays the current selection and manages opt-in/summary text.
