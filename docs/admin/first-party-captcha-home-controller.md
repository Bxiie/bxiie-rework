# Tenant CAPTCHA administration

Tenant public contact and signup forms use ArtsFolio's first-party human confirmation checkbox. No Google Cloudflare Turnstile domain registration is required for tenant subdomains or custom domains.

If a tenant form reports a CAPTCHA error, the message should render on the same branded page or in the enhanced form result area rather than sending visitors to an unbranded error page.
