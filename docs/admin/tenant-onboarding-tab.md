# Tenant Admin Onboarding

The **Onboarding** tab in the Tenant Admin sidebar is the permanent home for:

- the practical onboarding checklist;
- the guided new-admin tour;
- the tenant-admin function index;
- training-video resources;
- the onboarding reset switch.

Resetting onboarding deletes only checklist and guided-tour state. It does not
change artwork, public content, branding, users, domains, commerce, or billing.

The operational Dashboard no longer contains onboarding controls.
