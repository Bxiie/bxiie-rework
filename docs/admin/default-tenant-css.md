# Default tenant CSS

New tenants receive a documented snapshot of `public/assets/site.css`.

Before the snapshot is stored, provisioning:

- replaces legacy `Bxiie` references with `ArtsFolio`;
- removes every `/* End of file. */` marker;
- leaves the tenant additions section open for normal editing.

The shared public stylesheet is also kept neutral so future snapshots do not
inherit artist-specific branding.
