# Tenant custom CAPTCHA

ArtsFolio uses Cloudflare Turnstile for platform-domain forms and a built-in CAPTCHA for tenant websites.

Tenant administrators do not need to create Cloudflare widgets or manage CAPTCHA secrets. Tenant contact and email-list forms automatically render the built-in ArtsFolio challenge.

The Platform Admin Turnstile settings apply to ArtsFolio platform-domain forms only.
