# Artwork placement, password reset, and email branding

Tenant admins can use `/admin/artworks/placement` to assign artwork to the home page and portfolio sections.

Tenant admins can use `/admin/portfolio-sections/order` to drag artworks or edit sort numbers for each section, including the home page.

Password reset is available at `/password/forgot` and `/password/reset` on platform, tenant subdomain, and custom-domain hosts.

HTML email uses the public logo asset `/assets/logo_2.png`.
