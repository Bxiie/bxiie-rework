# Dashboards

## Platform Admin dashboard

The platform dashboard shows business and operations signals for ArtsFolio:

- tenant count and active tenant status
- paid-capable and complimentary tenants
- recent marketplace sales
- gross marketplace volume
- commission revenue
- seller net revenue
- background job health
- recent tenants
- active pricing plans and selling economics

Use this page as the first place to look for platform-level trouble: failed jobs, sales workflow activity, tenant growth, or pricing configuration drift.

## Tenant Admin dashboard

The tenant dashboard shows site-specific work:

- published artwork count
- draft artwork count
- for-sale artwork count
- low-stock multiple items
- recent traffic
- subscribers
- contact messages
- open sales orders
- current plan and sales readiness

The "Needs attention" table calls out work that is likely to matter next, such as draft artworks, unanswered messages, unshipped orders, or missing Stripe configuration.
