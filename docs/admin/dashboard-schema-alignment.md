# Dashboard and billing override notes

The platform dashboard shows tenant count, paid-capable tenant count, recent sales, recent tenants, plan selling controls, background jobs, and sales economics.

The tenant dashboard shows published work, for-sale work, traffic, mailing-list signups, contact messages, open sales orders, current plan, and next useful action.

The platform tenant detail page has a clearer billing override panel. Marking a tenant complementary waives only monthly platform service billing. Complementary tenants still pay platform commission and credit-card charges on sales.
