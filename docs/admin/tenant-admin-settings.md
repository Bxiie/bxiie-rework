# Tenant Admin Settings Administration

## Current editable settings

```text
site_title
site_admin_email
```

## Boundary

These are client/tenant settings, not platform settings.

Tenant admins should manage these values.

Platform settings should live separately under platform admin.

Turnstile keys or inherited platform protection
