# Admin notifications, watermarks, and table tools

Tenant administrators see red count badges beside **Messages** for records with status `new` and beside **Email Signups** for signups created since that list was last opened.

Watermarks are configured at **Tenant Admin → Settings → Watermark**. They are opt-in and affect public medium, large, and original artwork responses only. Stored files and administrator thumbnails remain unchanged.

Administrative tables now provide keyboard-accessible column sorting and a visible-row filter. Existing server-side filters and pagination still control the full result set.

Platform **System Operations** supports explicit UTC start/end dates, status filtering, and paging for recent checks.
