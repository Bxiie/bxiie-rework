# Platform admin routes

The platform admin sidebar links are expected to resolve on the platform domain, not tenant domains.

- `/admin` - dashboard.
- `/admin/tenants` - tenants.
- `/admin/domains` - domains.
- `/admin/pricing` - plan and pricing readout from the `plans` table.
- `/admin/platform-settings` - platform settings.
- `/admin/settings` - compatibility redirect to `/admin/platform-settings`.
- `/admin/stats` - platform rollup counts.
- `/admin/contact-messages` - latest contact messages across tenants.
- `/admin/email-outbox` - queued outbound email.
- `/admin/jobs` - background jobs.
- `/admin/workers` - worker heartbeats.
- `/admin/audit-log` - platform audit log.
- `/admin/routes` - route map.
