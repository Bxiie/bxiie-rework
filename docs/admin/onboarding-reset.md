# Resetting tenant onboarding

Tenant onboarding can be reset from either administrative surface.

## Tenant admin

Open the tenant dashboard, find **Onboarding** in the Workbench panel, and click **Reset onboarding**.

## Platform admin

Open **Tenants**, open the tenant, find **Onboarding**, and click **Reset onboarding**.

The action removes only onboarding, guided-tour, getting-started, and dashboard-checklist state. It does not alter artwork, content, events, users, domains, branding, billing, orders, or public-site configuration. Each reset is written to the audit log.
