# Custom public page slugs

Tenant Admin > Settings > Navigation allows an artist to change the public URL slugs for Portfolio, About, and Contact.

A value such as `Art Quilts` is normalized when saved to `art-quilts`, producing a public URL such as `/art-quilts`.

ArtsFolio also keeps `/portfolio`, `/about`, and `/contact` active for backward compatibility so existing bookmarks and indexed links do not break.

After changing a slug, open the public site from the navigation bar and verify both the page and any Contact form submission.
