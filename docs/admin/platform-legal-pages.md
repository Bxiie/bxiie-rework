# Platform legal pages

Platform admins should keep these public URLs available:

- `https://artsfol.io/terms`
- `https://artsfol.io/privacy`

Use these URLs for OAuth app configuration, Facebook data deletion references, Google app verification, support responses, footer links, and customer onboarding.

The privacy page tells users to send data deletion requests to `info@artsfol.io` with the subject `Data deletion request`, `Facebook data deletion request`, or `Google data deletion request` as appropriate.
