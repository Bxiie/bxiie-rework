# Platform UI, auth, spam, and stats settings

Platform admins manage global copyright, directory thumbnail size, OAuth provider IDs/secrets, reCAPTCHA defaults, and platform CSS from `/platform/admin/platform-settings`.

Tenant admins may override reCAPTCHA keys and artwork display ordering from `/admin/settings`. Contact and mailing-list submissions use tenant reCAPTCHA keys first and platform keys as fallback.

Platform stats show bar graphs for day and hour. The By Location table includes unique IP counts and a drilldown link showing hashed unique IPs and access counts for that location.
