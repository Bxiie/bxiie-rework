# ArtsFolio Video 01 production package

This package contains the complete production materials for:

**ArtsFolio Training 01: Your Admin Orientation**

Files:

- `01_admin_orientation_production_script.md`
  Full narration, screen direction, timing, title cards, and QA notes.
- `01_admin_orientation_teleprompter.txt`
  Narration only.
- `01_admin_orientation_shot_list.csv`
  Chapter-by-chapter recording plan.
- `01_admin_orientation_captions.srt`
  Ready-to-import subtitle file.
- `01_admin_orientation_publishing_metadata.md`
  Title, description, chapters, tags, and thumbnail brief.
- `01_admin_orientation_recording_checklist.md`
  Privacy, recording, editing, and release checks.

The package is designed around the Northstar Studio training tenant and current ArtsFolio navigation. It deliberately avoids requiring real customer, visitor, sales, or credential data.

No MP4 is included because this environment does not provide a browser screen recorder or voice-recording capability.
